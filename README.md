## 📌 프로젝트 소개
> **My Blockchain Project**는 블록체인 기술을 기반으로 한 간단한 P2P 네트워크입니다.  
> 노드 간 트랜잭션을 동기화하고 블록을 공유하는 기능을 제공합니다.

🔹 **주요 기능**
- ✅ P2P 네트워크 기반 블록체인
- ✅ 트랜잭션 처리 및 서명 검증
- ✅ 블록 생성 및 전파
- ✅ WebSocket을 활용한 실시간 동기화
